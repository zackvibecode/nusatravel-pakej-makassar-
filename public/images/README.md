# Images — Pakej Makassar

Letak gambar anda di dalam folder ini. Nama fail mesti sama seperti berikut supaya page automatik paparkan gambar tersebut:

| Fail | Saiz dicadang | Kegunaan |
|------|---------------|----------|
| `makassar-hero.jpg` | 1000×1250 (portrait 4:5) | Gambar utama hero section |
| `makassar-gallery-1.jpg` | 800×600 (4:3) | Gambar besar galeri |
| `makassar-gallery-2.jpg` | 600×600 (1:1) | Gambar kecil galeri |
| `makassar-gallery-3.jpg` | 600×600 (1:1) | Gambar kecil galeri |
| `makassar-gallery-4.jpg` | 600×600 (1:1) | Gambar kecil galeri |
| `makassar-gallery-5.jpg` | 600×600 (1:1) | Gambar kecil galeri |

## Tips
- Guna format `.jpg` untuk saiz fail yang kecil.
- Pastikan gambar berkualiti tinggi tapi tidak melebihi 500KB setiap satu.
- Boleh guna [squoosh.app](https://squoosh.app) untuk compress gambar.
- Jika tiada gambar, Next.js akan tunjuk ruang kosong (broken image) — jadi pastikan semua fail ada.
